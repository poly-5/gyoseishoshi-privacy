プライバシーポリシー

行政書士 合格アプリ（jp.ken.gyoseishoshi）

開発者 poly-5（以下「当方」）は、本アプリのご利用に際して取得する情報の取り扱いについて、以下のとおり定めます。

1. 収集する情報

当方は、以下の情報を収集する場合があります。

  • クラッシュレポート・パフォーマンスデータ — Expo / EAS（Expo Application Services）が提供する仕組みを通じて、アプリの異常終了やパフォーマンスに関する匿名の技術情報を収集します。これはアプリの品質改善を目的としたものです。
  • 学習進捗データ — 問題の正誤履歴や学習の進捗状況は、お使いの端末内にのみ保存されます。外部サーバーへの送信は行いません。

2. 収集しない情報

当方は、以下の情報を収集しません。

  • 氏名・メールアドレス・電話番号などの個人を特定できる情報
  • 位置情報（GPS・ネットワーク位置情報）
  • 連絡先・カメラ・マイクなどのセンシティブな端末データ

3. 第三者への提供

当方は、収集した情報を第三者に販売・共有・提供しません。ただし、Expo / EAS のサービス提供に必要な範囲で、当該サービスの利用規約に基づき技術情報が処理される場合があります。

4. 情報の保護

学習進捗データは端末上にローカル保存されるため、当方がサーバー側でアクセスすることはありません。クラッシュレポートは匿名化されて収集され、個人の特定には使用されません。

5. お問い合わせ

プライバシーポリシーに関するご質問・ご要望は、下記の GitHub Issues よりお知らせください。

https://github.com/poly-5/gyoseishoshi-privacy/issues

6. ポリシーの変更

本ポリシーは、必要に応じて予告なく改定することがあります。改定後のポリシーは本ページに掲載した時点で効力を生じるものとします。
